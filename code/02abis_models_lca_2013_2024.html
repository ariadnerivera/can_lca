Mapping the shifts: analysis of recreational cannabis policies from 2013-2024 in US states

Author

Ariadne Rivera

Background

Over the past two decades, rapid shifts have occurred in the legal landscape of cannabis. Many states have moved from a prohibitionist type of approach to legalization for medical use and towards the subsequent legalization of cannabis legalization for recreational use. As of November 2024, 39 states have permitted the use and commercialization of medical cannabis products, 9 states allowed the use low THC, high CBD products, and 24 states had legalized cannabis for recreational purposes (NCLS 2024). Consequently, approximately 80% of the population in the US resides in a county where they can access medical or adult-use cannabis through a retail outlet (Shah and Sono 2024).

This quarto file includes the final code to create figures and results from the models presented in the thesis.

# 1) a dummy point inside the panel so ggplot can draw a legend key for "RCL start"
dummy_pt <- with(lca4_sd, data.frame(
  year   = as.factor(sort(unique(year))[1]),

lca4_sd_gp <-
  ggplot(lca4_sd, aes(x = as.factor(year), y = stabbr,
                      fill = as.factor(lawtype_sd))) +
  geom_tile(color = "gray100") +
  
  # draw the rectangle outline in the plot, but NO legend from this layer
  geom_tile(
    data = rmlyear,
    aes(x = as.factor(year), y = stabbr),
    linewidth = 0.25, color = "black", fill = NA,
    inherit.aes = FALSE, show.legend = FALSE
  ) +
  
  # actual sales start squares (will provide one legend entry)
  geom_point(
    data = rmlslyear,
    aes(x = as.factor(year), y = stabbr, shape = "Sales start"),
    color = "black", size = 1.2, inherit.aes = FALSE
  ) +
  
  # legend-only entry for RCL start (invisible on plot, but appears in legend)
  geom_point(
    data = dummy_pt,
    aes(x = year, y = stabbr, shape = "RCL start"),
    color = "black", size = 0, alpha = 0, inherit.aes = FALSE
  ) +
  
  scale_y_discrete(limits = rev) +
  scale_fill_manual(values = cols,
                    labels = c("Class 1: No RCL","Class 2: Low Restrictions","Class 3: Moderate Restrictions","Class 4: High Restrictions"),
                    name   = "Latent Class") +
  
  # ONE combined legend block for both markers
  scale_shape_manual(
    name   = "",
    breaks = c("RCL start","Sales start"),
    values = c("RCL start" = 0,  # open square = law enacted
               "Sales start" = 15),  # filled square = sales start
    labels = c("RCL start"  = "RCL starts",
               "Sales start" = "RCL sales start")
  ) +
  
  guides(
    fill  = guide_legend(order = 1,
                         byrow = TRUE, 
                         keyheight = unit(10, "pt")),
    shape = guide_legend(order = 2,
                         byrow = TRUE,
                         override.aes = list(size = 3, color = "black", alpha = 1),
                         keyheight = unit(11, "pt"))
  ) +
  theme_minimal() +
  theme(
    legend.spacing.y  = unit(3, "pt"),       # no extra gap between rows
    legend.key.height = unit(10, "pt"),
    legend.box.spacing = unit(4, "pt")
  ) +
  labs(x = "Year", y = "State",
       title = "Standard LCA all states-years 2013–2024")
